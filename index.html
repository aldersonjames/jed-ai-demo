This is a web-based prototype of the Google Maps Narrator app. Each card below shows an actual iPhone screen from the app. You can interact with the embedded screens directly.

0. Landing Page

Welcome screen with app introduction, value proposition, and "Get Started" call-to-action. First impression for new users.

1. Import Route

Users paste their Google Maps URL to import an existing trip plan. The app will analyze the route and detect stops automatically.

2. Route Analysis

Shows the results of analyzing the imported route, including detected stops, categories, and route summary information.

3. Customize Stops

Users can enable/disable stops, add additional stops, and configure which points of interest to include in narration.

4. Narration Preview

Users can preview and test the narration content before starting their trip, including voice selection and audio controls.

5. Trip in Progress

Active trip screen showing current location, upcoming narration points, audio controls, and real-time progress tracking.

6. Drive-by Narration

Shows narration for points of interest that users pass by while driving, with contextual information and audio controls.

7. Settings

User preferences including voice selection, narration timing, interest categories, and audio settings.
